# SmartEdge EA — Changelog

All notable changes to **SmartEdge EA** will be documented here.

This project follows a structured versioning and update-log format to ensure
transparency and clarity for users, partners, and auditors.

---

## Version 1.0.0 — Initial Public Release
### Release Date: 2025-01-01
- Launch of SmartEdge EA (Multi-Currency Version)
- Core Strategy Components Added:
  - RSI-based entry logic
  - MACD optional confirmation
  - EMA trend filter
  - Grid-based position scaling
  - Average-price-based exit system
- Added USD-based risk control
- Added max-currency filter
- Added account stop-loss protection

---

## Version 1.1.0 — Enhanced Multi-Strategy Engine
### Release Date: 2025-02-10
- Introduced **multiple strategy sets (Set1–Set11)**
- Independent indicator configs per strategy
- Improved scaling logic with 2-step lot sizing
- Improved SymbolLock logic to prevent double trades
- Added Spread control per symbol

---

## Version 1.2.0 — License System & Validation
### Release Date: 2025-03-05
- Added MT4 license validation system
- Added account binding (MT4 account ↔ license)
- Introduced license expiration handling
- Improved validation error messages

---

## Version 1.3.0 — Stability Improvements

- Fixed initial trade placement edge-cases
- Optimized grid logic to reduce unnecessary scaling
- Improved handling of higher-timeframe EMA filters
- Several bug fixes & internal logging improvements
